# Simply Automated — Website

Next.js site for [simplyautomated.nz](https://simplyautomated.nz)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

Push to GitHub and connect to Vercel. Add `simplyautomated.nz` as a custom domain in Vercel settings.
